Attempts to cast the content type of a Box<[T]>.

On failure you get back an error along with the starting Box<[T]>.

Failure

  • The start and end content type of the Box<[T]> must have the exact same alignment.
  • The start and end content size in bytes of the Box<[T]> must be the exact same.
