Expand description

A segment of a circle.

If inner_radius > 0, then the shape will be a doughnut segment.

Fields

center: Point

The center.

outer_radius: f64

The outer radius.

inner_radius: f64

The inner radius.

start_angle: f64

The angle to start drawing the segment (in radians).

sweep_angle: f64

The arc length of the segment (in radians).
